Typescript

Typescript is een open source programmeertaal ontwikkeld en onderhouden door Microsoft. Het is een strikte superset van Javascript, dat wil zeggen dat alle valide JavaScript gelijk valide Typescript is (maar het omgekeerde is natuurlijk niet het geval).

De belangrijkste toevoeging bestaat uit (optionele) types (strict typing) en modularisatie (class-based programming). Hierdoor kunnen we variabele een type meegeven:

    let foo:string = "hallo"
    let done:boolean = false
    let getal:number = 314 // all numbers are floating point...
    let list:number[] = [3, 1, 4, 1, 5]
    let ding:any

    //Generics-achtige constructies
    let list2:Array<number> = [3.14, 2.18, 4.04]

    // Tuples
    let data:[string, number] = ["Henk", 12]
