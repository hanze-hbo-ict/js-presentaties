Command line TypeScript

Momenteel ondersteunt geen enkel browser native TypeScript. Dat is geen probleem, aangezien de voordelen ervan vooral compile time zijn. Het idee is dat je code schrijft in TypeScript, waar dan JavaScript van gemaakt wordt en dat dat naar de browser wordt gestuurd.

Om TypeScript op de command-line te runnen, kun je gebruik maken van ts-node. Dit doet feitelijk hetzelfde als wat je gewend bent met node.
