Modules (3/4)

In een module kun je ook meerdere klassen of functies exporteren. Daarvoor kun je verschillende syntaxen gebuiken. Allereerst de meest standaard JavaScript-versie:

  function foo() { ... }
  function bar() { ... }

  module.exports = { foo, bar }

In Typescript heb je behalve deze ook een wat meer verboze (en wat meer schaalbare) manier om hetzelfde te bereiken:

  function foo() { ... }
  function bar() { ... }

  export { foo, bar as BetereNaam }

Het importeren van meerdere functies of klassen gaat in alle gevallen op dezelfde manier:

  import { foo as LokaleNaam, BetereNaam } from './bestandsnaam'  // optie 1
  import { * } from './bestandsnaam' // optie 2
