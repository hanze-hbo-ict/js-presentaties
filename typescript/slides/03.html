class AwesomeClass {
    salutation:string

    constructor(naam:string) {
        this.salutation = naam
    }

    greet() {
        return "Hello, " + this.greeting
    }
}

let demo = new AwesomeClass("Henk");

Je ziet dat het sleutelwoord constructor wordt gebruikt, in plaats van de klasse-naam (zoals in Java). Verder moet je in Typescript de velden voorzien van this (wat in Java optioneel is).

Let op: classes worden in EcmaScript niet gehoist, zoals functies en (sommige) variabelen.
