Modules (4/4)

Binnen een module kun je ook één default export member (functie of klasse) hebben. Een dergelijke export voorzie je van het sleutelwoord default: 

  // bestand DEMO1.ts
  function foo() { ... }
  export default foo

Je kunt dat sleutelwoord ook direct bij het betreffende member zetten:

  // bestand DEMO2.ts
  // naam van de default klasse of de functie is optioneel
  export default function () { ... }

Het import-statement is ietsje anders wanneer dit een default member betreft:

  import foo from './DEMO1'
  import LokaleNaam from './DEMO2'
