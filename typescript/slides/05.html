Modules (1/4)

Sinds EcmaScript 2015 maakt JavaScript gebruik van Modules – en bij extensie wordt dit ook in Typescipt ondersteunt. Modules opereren binnen hun eigen scope, dus variabelen, functies en klassen zijn onzichtbaar buiten deze module, tenzij deze expliciet worden geëxporteerd.

Het importeren en exporteren van (onderdelen van) modules gebeurt op het niveau van het file system. De operationele kant hiervan wordt uitgevoerd door een module loader zoals require.js (voor browsers) of CommonJS (voor node).

Bestanden die geen export of import hebben, worden beschouwd als globale scripts waarvan de volledige inhoud beschikbaar is binnen de globale scope.
