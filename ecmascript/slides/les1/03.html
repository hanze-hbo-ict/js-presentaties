const

Een const is een block-scoped variabele waarvan je de waarde niet meer kunt veranderen wanneer je hem hebt ge√Ønitialiseerd.

{
  const a = 2
  console.log(a)
  a = 3; // TypeError: Assignment to constant variable.
}

De restrictie is op de assignment, niet op de waarde van de variabele:

{
  const a = [1,2,3]
  a.push(4)
  console.log(a); // [1 ,2, 3, 4]
  a = 3; // TypeError: Assignment to constant variable.
}
