for ... of loops

Behalve for en for...in heb je ook for ... of. Dit loopt over een itarable set van waarden:

var a = ["a", "b", "c", "d"]
for (var idx in a) { console.log(idx) } // 0, 1, 2, 3: 'normale' iteratie
for (var val of a) { console.log(val) } // 'a', 'b', 'c', 'd'

Op de achtergrond werkt for ... of met een iterator die herhaaldelijk wordt opgeroepen om de nieuwe waarde te produceren.

In EcmaScript zijn de volgende typen standaard iterable:

  • Arrays
  • Strings
  • Generators
  • Collections
