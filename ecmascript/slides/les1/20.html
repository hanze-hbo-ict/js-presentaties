Generators (1/4)

In EcmaScript kun je gebruik maken van generators: een functie waarvan de executie op een bepaald moment kan worden 'gepauzeerd' om op een later moment weer verder te gaan. Een dergelijke pauze kan gebruikt worden om comunicatie tussen de caller en de functie zelf op te zetten.

We geven een generator aan door middel van een asterisk (*):

function *foo() {
  // generator body
}

Het aanroepen van een generator gaat op een normale manier; maar de functie wordt op dat moment niet uitgevoerd, maar er wordt een iterator aangemaakt die de executie van die generator controlleerd:

function *foo(x, y) {  console.log(x, y) }

foo() // geen output; f kunnen we nu gebruiken als iterator
