Functies als default parameters

Hoewel het niet aan te raden is, kun je zelfs inline functie-aanroep gebruiken als standaardwaarde voor een parameter (een zogenaamde Immediately-invoked Function Expression, IIFE):

  function foo( x = (function(v){ return v + 11; })(31) ) {
    console.log(x)
  }
  foo(); // 42

Iets vergelijkbaars kan nuttig zijn als je een no-op functie als default callback wilt hebben: 

  function ajax(url, cb=function(){}) {
    //
  }
