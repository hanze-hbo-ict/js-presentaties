Arrow functions (1/4)

Net als in Java (vanaf versie 8) kennen we in EcmaScript anonieme functie-definies. In EcmaScript kun je een dergelijke functie aan een variabele toekennen: 

  // normal function:
  function foo(x, y) {
    return x + y
  }

  //arrow function:
  var foo = (x, y) => x + y

Een arrow-functie bestaat uit een parameterlijst van nul of meer parameters, omgeven door haakjes als er niet exact één parameter is. Hierna komt de arrow (=>; let op: in Java is dit ->) en daarna de method body. Als deze body uit één regel bestaat, hoeft deze niet te worden voorzien van accolades en wordt er impliciet een return voor deze regel gezet.
