Arrow functions (3/4)

Arrow-functies zijn altijd functie-expressies; er bestaat geen arrow-functie declaratie. Het zijn anonieme expressies â€“ er is geen expliciete referentie die je kunt gebruiken voor event-binding of recursie.

Ze verschillen ook van Immediately-invoked function expressions (IIFE) omdat deze laatste direct uitgevoerd worden.

(function foo() { console.log('dit wordt uitgevoerd') })();
() => console.log('dit wordt NIET uitgevoerd');
( () => console.log('dit wordt dan weer WEL uitgevoerd') )()

Om deze voorbeelden te laten werken is het noodzakelijk dat de functie-definities dit keer wel met een puntkomma wordt afgesloten. Het voert te ver om dit toe toe lichten; het heeft te maken met intermediate values binnen IIFE.
