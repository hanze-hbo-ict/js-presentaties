Arrow functions (4/4)

Het gaat er bij arrow-functies niet om minder te typen. De belangrijkste reden voor de toevoeging van deze functionaliteit aan de taal heeft te maken met binding.

var foo = {
  handleRequest: function() {
    var self = this;
    btn.addEventListener('click', function(){
      self.handleRequest();
    })
  }
}

De hack hierboven self=this wordt gebruikt om het object waarin de functie handleRequest gedefinieerd is over te brengen naar de scope van de anonieme functie van addEventListener.
