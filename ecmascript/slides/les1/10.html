Destructuring objects (1/2)

We kunnen dezelfde constructie gebruiken wanneer een functie een object teruggeeft. In dat geval kunnen we gebruik maken van de key-value pairs uit dat object:

function foo() {
  return { x:4, y:5, z:6 }
}

let { x:x, y:y, z:z } = foo()
console.log(y); // 5

Als de naam van de variabele hetzelfde is als de property van het object, kun je ook de kortere syntax gebruiken:

let { x, y, z } = foo()
console.log(z); // 6
