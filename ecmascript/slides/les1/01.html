let en var(1/2)

EcmaScript kent twee soorten variabele-definities: let en var. Het grote verschil zit hem in de scoping.

    var foo = "hallo"
    for (let i=0; i<10; i++){
      console.log(foo)
    }
    console.log(i); // reference error: i is not defined

Terwijl

    var foo = "hallo"
    for (var i=0; i<10; i++){
      console.log(foo)
    }
    console.log(i); // 10
