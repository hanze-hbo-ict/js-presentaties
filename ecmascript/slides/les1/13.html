Template literals (or interpolated strings)

Je kunt gebruik maken van de backtick (`) om variabele binnen die string om te zetten in de waarde van die variabele op dat moment.

let name = 'Henk de Boer'
let groet = `Hallo ${name}`
console.log(groet) //Hallo Henk de Boer

Deze constructie stelt je ook in staat string over meerdere regels te definiëren – de newlines worden in dat geval in de output gehandhaafd. 

var txt = `Wilhelmus van Nassau
Ben ik van duitschen bloed.`
console.log(txt); // output is twee regels lang
