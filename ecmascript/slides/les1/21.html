Generators (2/4)

Een generator geeft een waarde terug en pauzeert dan de executie totdat door de client een volgende waarde wordt gevraagd. Hiervoor is het sleutelwoord yield.

Een yield geeft een object terug met een waarde en de property done; aan de hand van deze property kun je checken of de generator is 'uitgeyield':

function *foo(){
  yield 20
  yield 30
}

let f = foo()
console.log(f.next()) // { value: 20, done: false }
console.log(f.next()) // { value: 30, done: false }
console.log(f.next()) // { value: undefined, done: true }
