De ...-operator: spread en rest

De exacte naam van de ...-operator is afhankelijk van waar en hoe deze wordt ingezet.

    function foo(x,y,z) {
      console.log(x,y,z);
    }
    foo (...[1,2,3]); // 1 2 3

De ... in de functie-aanroep zorgt ervoor dat de array wordt uitgespreid naar de individuele waarden. Dit kan in allerlei contexten:

    var a = [2,3,4]
    var b = [1, ...a, 5]; // [1, 2, 3, 4, 5]
