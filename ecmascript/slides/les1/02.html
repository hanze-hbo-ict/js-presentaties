let en var (2/2)

var's worden gehoist, let's niet: 

{
      console.log(a); // undefined
      console.log(b); // Reference Error

      var a;
      let b;
}

Vanzelfsprekend blijft er een verschil tussen initialisatie en declaratie.

{
  console.log(b); // undefined, maar niet Reference Error
  var b
  b = "hallo"
}
