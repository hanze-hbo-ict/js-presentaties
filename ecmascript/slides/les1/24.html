Promises

Promises zijn vergelijkbaar met callback-functions of met event-listeners. Ze vormen een intermediair tussen de caller en de async uitgevoerde code. Je kunt het ook zien als een future value.

Een Promise wordt altijd maar één keer uitgevoerd (resolved), en de de uitkomst hiervan is altijd ofwel fullfilled ofwel rejected.

Het maken van een Promise is behoordelijk elementair:

var prom = new Promise (function (resolve, reject) {
  if (alles_ok) resolve (return_value)
  else reject (error)
})

prom
  .then( return_value => function )
  .catch( error => function )
