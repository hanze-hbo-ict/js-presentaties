Generators (4/4)

Een yield statement is geen operator, hoewel hij er wel op lijkt. Hij heeft dezelfde expression presendence als een assigment, wat inhoudt dat yield legaal is overal waar bijvoorbeeld a = 3 legaal is:

function *foo() {
  var arr = [ yield 11, yield 13, yield 17]
  yield arr
}

var f = foo()

console.log(f.next().value)    // 11
console.log(f.next(314).value) // 13
console.log(f.next(218).value) // 17
console.log(f.next(42).value)  // [ 314, 218, 42 ]
