Default parameter waarden

Net als in php en Python kun je in JavaScript standaardwaarden voor parameters meegeven in de signature van de functie:

  function foo(x=11, y=31){
    return (x + y)
  }
  console.log( foo() );       // 42
  console.log( foo(2) );      // 33
  console.log( foo(20, 30) ); // 50

Omdat functies in JavaScipt first class citizens zijn, kun je deze ook gebruiken als standaardwaarde voor een parameter:

  function bar(x=3, y=foo(5)) {
    console.log( x * y );
  }
  bar();     // 108
  bar(3, 4); // 12
